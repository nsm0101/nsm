# Pass The Pointer 2026 — Additional Pediatric POCUS Topics

Supplemental teaching slides built in the **Pass The Pointer 2026** template
style, covering five high-yield pediatric point-of-care ultrasound (POCUS)
applications. Designed to be dropped into the master deck.

## What's here

- **`Pass_the_Pointer_2026_NEW_TOPICS_only.pptx`** — the new slides only
  (77 slides), in the template's layouts/theme. Small and easy to merge into
  the master deck.
- A full merged deck (master template + these new topics, ~30 MB) was also
  produced and delivered separately; it is intentionally kept out of the
  website repo to avoid bloat.

## Topics covered

1. **Small Bowel Obstruction** — keyboard/valvulae sign, dilated loops >2.5 cm, peristalsis
2. **Intussusception** — target (transverse) + pseudokidney (longitudinal) signs
3. **Femur / Long-Bone Fracture** — cortical step-off, periosteal hematoma, 2-plane scanning
4. **Pneumothorax** — lung sliding, seashore vs barcode (M-mode), lung point
5. **Constipation** — transrectal diameter (TRD); cutoff **≥ 3.8 cm**

## Slide structure (per topic)

Section divider → Case → Indications → Set-Up (probe/preset/position) →
Technique → Normal anatomy → Pathology → Pearls & Pitfalls.

- **Speaker notes** are written on **every** slide.
- A **blank "Insert Image" slide follows each teaching slide** so you can drop
  in your own cine clips / stills.
- Representative **normal-vs-pathology ultrasound images** are embedded on the
  anatomy/findings slides (credits below). The femur module uses a labeled
  schematic because no openly-licensed long-bone-fracture ultrasound still was
  available.
